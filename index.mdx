  <h2 className="text-3xl font-bold text-gray-900 dark:text-white mb-4">
    Quick Start
  </h2>
  <p className="text-gray-600 dark:text-gray-400 mb-8">
    Get up and running with Eru in minutes
  </p>

  <Steps>
    <Step title="Download and install">
      Download Eru for your platform (macOS, Windows, or Linux) from the [releases page](https://github.com/royalbhati/eru/releases/latest).
      
      For macOS users, you may need to bypass Gatekeeper:
      ```bash
      xattr -c /Applications/Eru.app
      ```
    </Step>

    <Step title="Create your first request">
      Launch Eru and create a new request. Set the method (GET, POST, etc.), enter your URL, and add any headers or body data you need.

      ```json
      {
        "method": "GET",
        "url": "https://api.example.com/users",
        "headers": {
          "Authorization": "Bearer your-token"
        }
      }
      ```
    </Step>

    <Step title="Organize with collections">
      Group related requests into collections for better organization. Collections can be synced with Git for team collaboration.
    </Step>

    <Step title="Use environment variables">
      Create environments (dev, staging, production) and define variables to switch contexts easily without editing requests.
    </Step>
  </Steps>
</div>
